## 🌊 Overview
**OceanWatch** is a comprehensive web platform dedicated to the conservation and sustainable use of marine resources. Aligned with **United Nations Sustainable Development Goal 14 (Life Below Water)**, the project serves as an advocacy hub to raise awareness about marine ecosystem challenges and empower communities to take action.

### 🚀 Key Features
* **Educational Resources**: In-depth information regarding SDG 14 and the protection of marine life.
* **Publication Library**: Direct access to global reports, including "Synergy Solutions for Climate and SDG Action" and "Partnership Snapshots".
* **Volunteer Portal**: A functional "Get Involved" section allowing users to register and join conservation efforts.
* **Responsive Design**: A fully optimized experience across desktop and mobile devices using modern CSS and Bootstrap.
